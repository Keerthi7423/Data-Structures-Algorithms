========================================
JAVASCRIPT JSON NOTES
========================================

----------------------------------------
WHAT IS JSON?
----------------------------------------
JSON = JavaScript Object Notation

- Used to store and transfer data
- Mostly used in APIs
- Looks like JavaScript object

----------------------------------------
EXAMPLE JSON
----------------------------------------
{
    "name": "Keerthi",
    "age": 22,
    "city": "Bangalore"
}

----------------------------------------
KEY DIFFERENCE (OBJECT vs JSON)
----------------------------------------

JavaScript Object:
let obj = {
    name: "Keerthi"
};

JSON:
{
    "name": "Keerthi"
}

Differences:
- JSON keys MUST be in double quotes ""
- JSON is string format (text)
- Object is real JavaScript structure

----------------------------------------
JSON.stringify()
----------------------------------------

Convert object → JSON string

let obj = {name:"Keerthi"};

let json = JSON.stringify(obj);

console.log(json);

Output:
'{"name":"Keerthi"}'

----------------------------------------
JSON.parse()
----------------------------------------

Convert JSON string → object

let json = '{"name":"Keerthi"}';

let obj = JSON.parse(json);

console.log(obj.name);

Output:
Keerthi

----------------------------------------
WHY JSON IS USED?
----------------------------------------

- API communication
- Data storage
- Sending data from server to client
- Lightweight and fast

----------------------------------------
REAL WORLD FLOW
----------------------------------------

Server → sends JSON  
Frontend → parses JSON → uses data  

----------------------------------------
COMMON MISTAKES
----------------------------------------

❌ Single quotes in JSON
❌ Missing quotes for keys
❌ Trailing commas

----------------------------------------
VALID JSON RULES
----------------------------------------

✔ Keys must be in double quotes  
✔ Strings must use double quotes  
✔ No functions allowed  
✔ No undefined  

----------------------------------------
SUMMARY
----------------------------------------

JSON = data format  
stringify → object → JSON  
parse → JSON → object  

----------------------------------------
END

========================================
JSON PRACTICE
QUESTIONS + ANSWERS + EXPLANATION
========================================

----------------------------------------
🟢 BASIC
----------------------------------------

1) Convert object → JSON
----------------------------------------
Question:
Convert JS object to JSON

Code:
let obj = {name: "Keerthi", age: 22};

let json = JSON.stringify(obj);
console.log(json);

Answer:
'{"name":"Keerthi","age":22}'

Explanation:
- JSON.stringify() converts object → string
- Used when sending data to server

----------------------------------------

2) Convert JSON → object
----------------------------------------
Question:
Convert JSON string to object

Code:
let json = '{"name":"Keerthi","age":22}';

let obj = JSON.parse(json);
console.log(obj);

Answer:
{ name: "Keerthi", age: 22 }

Explanation:
- JSON.parse() converts string → object
- Now we can access properties

----------------------------------------

3) Access value after parsing
----------------------------------------
Code:
let json = '{"name":"Keerthi","age":22}';

let obj = JSON.parse(json);

console.log(obj.name);

Answer:
Keerthi

Explanation:
- After parsing, it becomes normal object
- Access using dot notation

----------------------------------------
🟡 INTERMEDIATE
----------------------------------------

4) Convert array of objects to JSON
----------------------------------------
Code:
let arr = [
    {name:"A"},
    {name:"B"}
];

let json = JSON.stringify(arr);
console.log(json);

Answer:
'[{"name":"A"},{"name":"B"}]'

Explanation:
- Entire array converted into JSON string

----------------------------------------

5) Parse JSON and print all values using loop
----------------------------------------
Code:
let json = '{"a":10,"b":20,"c":30}';

let obj = JSON.parse(json);

for(let key in obj){
    console.log(obj[key]);
}

Answer:
10
20
30

Explanation:
- for...in loop accesses keys
- obj[key] gives value

----------------------------------------

6) Modify parsed object
----------------------------------------
Code:
let json = '{"name":"Keerthi"}';

let obj = JSON.parse(json);

obj.name = "Kumar";

console.log(obj);

Answer:
{name: "Kumar"}

Explanation:
- Parsed object behaves like normal object
- We can update values

----------------------------------------
🔴 LOGIC BUILDING
----------------------------------------

7) Count number of keys in parsed JSON
----------------------------------------
Code:
let json = '{"a":1,"b":2,"c":3}';

let obj = JSON.parse(json);

let count = Object.keys(obj).length;

console.log(count);

Answer:
3

Explanation:
- Object.keys() returns array of keys
- length gives count

----------------------------------------

8) Find max value from JSON object
----------------------------------------
Code:
let json = '{"a":10,"b":50,"c":30}';

let obj = JSON.parse(json);

let max = -Infinity;

for(let key in obj){
    if(obj[key] > max){
        max = obj[key];
    }
}

console.log(max);

Answer:
50

Explanation:
- Traverse all values
- Track maximum

----------------------------------------

9) Convert object to JSON and store
----------------------------------------
Code:
let obj = {city:"Bangalore"};

let jsonData = JSON.stringify(obj);

console.log(jsonData);

Answer:
'{"city":"Bangalore"}'

Explanation:
- Stored JSON string in variable

----------------------------------------

10) Print only string values
----------------------------------------
Code:
let json = '{"a":"hi","b":10,"c":"js"}';

let obj = JSON.parse(json);

for(let key in obj){
    if(typeof obj[key] === "string"){
        console.log(obj[key]);
    }
}

Answer:
hi
js

Explanation:
- typeof checks data type
- Only strings printed
