----------------------------------------
1) STRING
----------------------------------------
A string is a data type used to store text.

Example:
let name = "Keerthi";

----------------------------------------
2) CREATE STRING
----------------------------------------
Strings can be created using:
- Double quotes (" ")
- Single quotes (' ')
- Backticks (` `)

----------------------------------------
3) TEMPLATE LITERALS
----------------------------------------
Template literals use backticks and allow embedding variables.

Example:
let name = "Keerthi";
console.log(`Hello ${name}`);

----------------------------------------
4) STRING IMMUTABILITY
----------------------------------------
Strings cannot be changed after creation.

Example:
let str = "Hello";
str[0] = "h"; → no change

----------------------------------------
5) STRING LENGTH
----------------------------------------
.length property is used to find length.

Example:
let str = "Hello";
console.log(str.length); → 5

----------------------------------------
6) ACCESS CHARACTERS
----------------------------------------
Using index:

Example:
let str = "JavaScript";
console.log(str[0]); → J

----------------------------------------
7) ESCAPE CHARACTERS
----------------------------------------
Used for special characters.

Examples:
\" → double quote
\n → new line
\t → tab

----------------------------------------
