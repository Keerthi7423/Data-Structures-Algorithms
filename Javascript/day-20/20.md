========================================
ARRAY METHODS
QUESTIONS + ANSWERS + EXPLANATION
========================================

----------------------------------------
1) push()
----------------------------------------
Question:
Add 100 to end of array

Code:
let arr = [1, 2, 3];
arr.push(100);

Answer:
[1, 2, 3, 100]

Explanation:
- push() adds element at the end
- It modifies original array
- Fast operation (O(1))

----------------------------------------
2) pop()
----------------------------------------
Question:
Remove last element

Code:
let arr = [10, 20, 30];
arr.pop();

Answer:
[10, 20]

Explanation:
- pop() removes last element
- Returns removed value (30)

----------------------------------------
3) unshift()
----------------------------------------
Question:
Add element at beginning

Code:
let arr = [2, 3];
arr.unshift(1);

Answer:
[1, 2, 3]

Explanation:
- Adds element at index 0
- Shifts all elements right
- Slower than push (O(n))

----------------------------------------
4) shift()
----------------------------------------
Question:
Remove first element

Code:
let arr = [5, 6, 7];
arr.shift();

Answer:
[6, 7]

Explanation:
- Removes first element
- Shifts remaining elements left

----------------------------------------
5) join()
----------------------------------------
Question:
Convert array to string

Code:
let arr = ["JS", "is", "fun"];
arr.join(" ");

Answer:
"JS is fun"

Explanation:
- Joins elements into string
- Uses separator (space here)

----------------------------------------
6) map() - double values
----------------------------------------
Question:
Double all numbers

Code:
let arr = [1, 2, 3];
let result = arr.map(num => num * 2);

Answer:
[2, 4, 6]

Explanation:
- map() creates new array
- Each element is transformed

----------------------------------------
7) map() - uppercase
----------------------------------------
Question:
Convert all strings to uppercase

Code:
let arr = ["a", "b", "c"];
let result = arr.map(str => str.toUpperCase());

Answer:
["A", "B", "C"]

Explanation:
- Each string converted individually
- Original array unchanged

----------------------------------------
8) map() - add 5
----------------------------------------
Question:
Add 5 to each number

Code:
let arr = [10, 20, 30];
let result = arr.map(num => num + 5);

Answer:
[15, 25, 35]

Explanation:
- Transformation applied to each element

9) map() - square
----------------------------------------
Question:
Square each number

Code:
let arr = [2, 3, 4];
let result = arr.map(num => num * num);

Answer:
[4, 9, 16]

10) filter() - even numbers
----------------------------------------
Question:
Get even numbers

Code:
let arr = [1, 2, 3, 4];
let result = arr.filter(num => num % 2 === 0);

Answer:
[2, 4]

Explanation:
- filter() returns elements that match condition
- Removes unwanted values

----------------------------------------
11) filter() - >20
----------------------------------------
Question:
Get numbers greater than 20

Code:
let arr = [10, 25, 30, 5];
let result = arr.filter(num => num > 20);

Answer:
[25, 30]

----------------------------------------
12) filter() - remove negatives
----------------------------------------
Question:
Remove negative numbers

Code:
let arr = [-2, 5, -1, 10];
let result = arr.filter(num => num >= 0);

Answer:
[5, 10]

13) find()
----------------------------------------
Question:
Find first number > 10

Code:
let arr = [5, 8, 12, 20];
let result = arr.find(num => num > 10);

Answer:
12

Explanation:
- Returns first matching value only

----------------------------------------
14) findIndex()
----------------------------------------
Question:
Find index of 30

Code:
let arr = [10, 20, 30, 40];
let result = arr.findIndex(num => num === 30);

Answer:
2

----------------------------------------
15) some()
----------------------------------------
Question:
Check if any number is negative

Code:
let arr = [1, 2, -3];
let result = arr.some(num => num < 0);

Answer:
true

Explanation:
- Returns true if ANY element matches

----------------------------------------
16) every()
----------------------------------------
Question:
Check if all numbers are positive

Code:
let arr = [1, 2, 3];
let result = arr.every(num => num > 0);

Answer:
true

Explanation:
- Returns true only if ALL match

----------------------------------------
17) reduce() - sum
----------------------------------------
Question:
Find sum of array

Code:
let arr = [1, 2, 3, 4];
let sum = arr.reduce((acc, num) => acc + num, 0);

Answer:
10

Explanation:
- acc = accumulator
- Starts from 0
- Adds each value

----------------------------------------
18) reduce() - max
----------------------------------------
Question:
Find maximum number

Code:
let arr = [10, 5, 25, 8];
let max = arr.reduce((a, b) => a > b ? a : b);

Answer:
25

----------------------------------------
19) reduce() - total string length
----------------------------------------
Question:
Find total characters

Code:
let arr = ["hi", "hello"];
let total = arr.reduce((sum, str) => sum + str.length, 0);

Answer:
7

----------------------------------------
FINAL SUMMARY
----------------------------------------

push → add end
pop → remove end
unshift → add start
shift → remove start

map → transform each element
filter → select elements
find → first matching value
some → any match
every → all match
reduce → combine into one value
