========================================
CALLBACK FUNCTIONS - NOTES
========================================

----------------------------------------
WHAT IS CALLBACK?
----------------------------------------
A callback is a function passed as an argument
to another function and executed later.

----------------------------------------
BASIC EXAMPLE
----------------------------------------
function greet(name, callback) {
    console.log("Hello " + name);
    callback();
}

function done() {
    console.log("Done!");
}

greet("Keerthi", done);

----------------------------------------
OUTPUT
----------------------------------------
Hello Keerthi
Done!

----------------------------------------
WHY CALLBACKS?
----------------------------------------

- To control execution order
- To reuse logic
- Used heavily in async operations

----------------------------------------
INLINE CALLBACK
----------------------------------------

greet("Keerthi", function() {
    console.log("Inline callback");
});

----------------------------------------
ARROW CALLBACK
----------------------------------------

greet("Keerthi", () => {
    console.log("Arrow callback");
});

----------------------------------------
REAL USE CASE
----------------------------------------

setTimeout(() => {
    console.log("Runs after 2 seconds");
}, 2000);

----------------------------------------
IMPORTANT POINT
----------------------------------------

Callback is NOT called immediately,
it is passed and executed later.

----------------------------------------
COMMON MISTAKE
----------------------------------------

❌ Wrong:
greet("Keerthi", done());

✔ Correct:
greet("Keerthi", done);

----------------------------------------
SUMMARY
----------------------------------------

callback = function passed to another function
executed later

----------------------------------------
END
----------------------------------------


========================================
CALLBACK PRACTICE
========================================

----------------------------------------
1) Simple callback
----------------------------------------
function process(num, cb) {
    return cb(num);
}

function double(x) {
    return x * 2;
}

console.log(process(5, double));

Answer:
10

----------------------------------------

2) Arrow callback
----------------------------------------
let process = (num, cb) => cb(num);

console.log(process(5, x => x + 10));

Answer:
15

----------------------------------------

3) Print after function
----------------------------------------
function task(cb) {
    console.log("Task running...");
    cb();
}

task(() => console.log("Task finished"));

Answer:
Task running...
Task finished

----------------------------------------

4) Callback with condition
----------------------------------------
function check(num, cb1, cb2) {
    if (num % 2 === 0) {
        cb1();
    } else {
        cb2();
    }
}

check(4,
    () => console.log("Even"),
    () => console.log("Odd")
);

Answer:
Even

----------------------------------------

5) Callback in loop
----------------------------------------
function processArray(arr, cb) {
    for (let i = 0; i < arr.length; i++) {
        cb(arr[i]);
    }
}

processArray([1,2,3], x => console.log(x * 2));

Answer:
2
4
6
