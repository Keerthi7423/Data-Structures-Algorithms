----------------------------------------
1) WHAT IS TYPE CONVERSION?
----------------------------------------
Type conversion is the process of converting one data type into another.

----------------------------------------
2) TYPES OF TYPE CONVERSION
----------------------------------------
- Implicit (automatic)
- Explicit (manual)

----------------------------------------
3) IMPLICIT vs EXPLICIT
----------------------------------------
Implicit:
Done automatically by JavaScript

Example:
"5" + 2 → "52"

----------------------------------------

Explicit:
Done manually by developer

Example:
Number("5") → 5

----------------------------------------
4) WHAT IS TYPE COERCION?
----------------------------------------
Type coercion is automatic type conversion by JavaScript during operations.

----------------------------------------
5) STRING TO NUMBER
----------------------------------------
Number("100") → 100

----------------------------------------
6) NUMBER TO STRING
----------------------------------------
String(50) → "50"

----------------------------------------
7) BOOLEAN()
----------------------------------------
Boolean() converts a value to true or false

Example:
Boolean(0) → false
Boolean(1) → true

----------------------------------------
OUTPUT QUESTIONS
----------------------------------------

8) console.log("5" + 2)
Answer: "52"
Reason: string concatenation

----------------------------------------

9) console.log("5" - 2)
Answer: 3
Reason: string converted to number

----------------------------------------

10) console.log("5" * "2")
Answer: 10
Reason: both converted to number

----------------------------------------

11) console.log(true + 1)
Answer: 2
Reason: true = 1

----------------------------------------

12) console.log(false + 1)
Answer: 1
Reason: false = 0

----------------------------------------

13) console.log("10" / 2)
Answer: 5
Reason: string converted to number

----------------------------------------

14) console.log("Hello" - 2)
Answer: NaN
Reason: invalid number conversion

----------------------------------------

15) console.log(Number("123"))
Answer: 123

----------------------------------------

16) console.log(Number("abc"))
Answer: NaN

----------------------------------------

17) console.log(Boolean(0))
Answer: false

----------------------------------------

18) console.log(Boolean(1))
Answer: true

----------------------------------------
BONUS (TRICKY)
----------------------------------------

24) console.log([] + [])
Answer: ""
Reason: empty strings combined

----------------------------------------

25) console.log([] + {})
Answer: "[object Object]"

----------------------------------------

26) console.log({} + [])
Answer: 0 (or "[object Object]" depending on context)

----------------------------------------

27) console.log(null + 1)
Answer: 1

----------------------------------------

28) console.log(undefined + 1)
Answer: NaN

----------------------------------------
FINAL SUMMARY
----------------------------------------

- "+" → string conversion
- "-", "*", "/" → number conversion
- true = 1, false = 0
- null = 0, undefined = NaN
