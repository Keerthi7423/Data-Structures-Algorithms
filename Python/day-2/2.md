QUESTION 2
--------------------------------------------------
What is the difference between global variables and local variables?

Answer:
Global Variables:
Variables declared outside a function and can be accessed anywhere in the program.

Local Variables:
Variables declared inside a function and can only be used within that function.

Example:

x = 10   # global variable

def test():
    y = 5   # local variable

QUESTION 5
--------------------------------------------------
What are the rules for naming variables in Python?

Answer:

1. Variable name must start with a letter or underscore.
2. Variable name cannot start with a number.
3. Variable name can contain letters, numbers, and underscores.
4. Variable names are case-sensitive.
5. Variable name cannot be a Python keyword.
