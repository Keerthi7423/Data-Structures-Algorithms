QUESTION 1
--------------------------------------------------
What is string concatenation?

Answer:
String concatenation is the process of combining two or more strings into one string using operators like +.

Example:
a = "Hello"
b = "World"
print(a + b)


--------------------------------------------------
QUESTION 2
--------------------------------------------------
What are different ways to combine strings in Python?

Answer:
1. Using + operator
2. Using f-strings
3. Using .format() method
4. Using join() method


--------------------------------------------------
QUESTION 3
--------------------------------------------------
What is an f-string?

Answer:
An f-string is a way to format strings by embedding variables directly inside {} using the prefix f.

Example:
name = "Keerthi"
print(f"My name is {name}")


--------------------------------------------------
QUESTION 4
--------------------------------------------------
Difference between concatenation and f-string

Answer:
Both produce same output.
Concatenation uses + operator.
f-strings are more readable and modern.
